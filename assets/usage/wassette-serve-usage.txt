Begin handling requests over the specified protocol

Usage: wassette serve [OPTIONS]

Options:
      --plugin-dir <PLUGIN_DIR>  Directory where plugins are stored. Defaults to $XDG_DATA_HOME/wasette/components
      --stdio                    Enable stdio transport
      --sse                      Enable SSE transport
      --streamable-http          Enable streamable HTTP transport
      --env <ENV_VARS>           Set environment variables (KEY=VALUE format). Can be specified multiple times
      --env-file <ENV_FILE>      Load environment variables from a file (supports .env format)
  -h, --help                     Print help
